## break command

break [N]<br>
// N is the number of nested loops.<br>
// This parameter is optional.<br>
// By default the value of N is 1.<br>

## continue command

continue [N]<br>
// the optional parameter N specifies the nth enclosing loop to continue from.<br>
// This parameter is optional.<br>
// By default the value of N is 1.<br>
